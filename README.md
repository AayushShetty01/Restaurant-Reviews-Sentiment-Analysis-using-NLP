# Restaurant Reviews Sentiment Analysis

An NLP-based sentiment analysis system built using Python to extract insights from restaurant reviews.

## Features
- Text preprocessing using NLTK
- TF-IDF feature extraction
- Logistic Regression and SVM classifiers
- 78% accuracy on sentiment classification
- Modular and Git-friendly codebase

## Tech Stack
Python, Pandas, NLTK, Scikit-learn

## How to Run
pip install -r requirements.txt
python src/train.py
